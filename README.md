# Kalkulator 

Kalkulator interaktif ini dibuat menggunakan HTML, CSS (TailwindCSS), dan JavaScript. Kalkulator ini memiliki berbagai fitur, termasuk memory functions, tema toggle, dan keyboard shortcuts untuk memudahkan pengguna dalam melakukan perhitungan matematika.

## Fitur Kalkulator:
### 1. Tombol MC, MR, M+, dan M-
   - MC (Memory Clear): Menghapus nilai yang tersimpan di dalam memori.
   - MR (Memory Recall): Menampilkan nilai yang tersimpan di dalam memori pada layar kalkulator.
   - M+ (Memory Plus): Menambahkan nilai yang sedang ditampilkan pada layar ke dalam memori.
   - M- (Memory Minus): Mengurangi nilai yang sedang ditampilkan pada layar dari memori.

### 2. Tombol Tema Toggle:
   - Toggle Switch digunakan untuk beralih antara dua tema:
     - Mode Pink (default): Tema dengan latar belakang putih dan tombol berwarna pink.
     - Mode Clean: Tema dengan tombol berwarna putih dan border tipis.
   - Klik toggle untuk mengganti tema antara Mode Pink dan Mode Clean.

### 3. Shortcut Keyboard:
   - Angka: Tekan angka dari 0 hingga 9 untuk memasukkan nilai.
   - Operasi Matematika: Tekan tombol berikut untuk operasi:
     - (+) untuk penjumlahan (klik +)
     - (-) untuk pengurangan (klik -)
     - ร untuk perkalian (klik *)
     - รท untuk pembagian (klik /)
   - Tombol Del : Tekan Backspace untuk menghapus karakter terakhir yang dimasukkan.
   - Tombol C: Tekan Esc untuk menghapus seluruh ekspresi.
   - Tombol CE: Tekan Delete untuk menghapus input terakhir.
   - Tombol Equals (=): Tekan Enter untuk menghitung hasil dari ekspresi matematika.
   - Tombol Dot (.): Tekan . untuk memasukkan tanda desimal.

### 4. Penyimpanan Memori:
   - Kalkulator memungkinkan penyimpanan nilai perhitungan dalam memori.
   - M+ dan M- untuk menambah atau mengurangi nilai memori.
   - MR untuk menampilkan nilai memori di layar kalkulator.

## Ringkasan Fitur:
- Tema Toggle: Mengubah tampilan antara dua mode (Mode Pink dan Mode Clean).
- Memory Buttons:
  - MC: Menghapus nilai memori.
  - MR: Memanggil nilai dari memori.
  - M+: Menambah nilai layar ke dalam memori.
  - M-: Mengurangi nilai layar dari memori.
- Keyboard Shortcuts: Menggunakan keyboard untuk memasukkan angka, operasi, dan tombol lainnya.
- Responsif Desktop dan Mobile
